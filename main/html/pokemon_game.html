ポケモン風ゲーム紹介

タイトル画面

モザイク加工された背景と「TAP TO START!」の演出で、冒険の始まりを演出。 BGM付きで没入感を高めました。

バトル開始画面

「強敵のガブリアスが現れた！」と登場演出があり、HPバーと技選択ボタンを表示してバトルの緊迫感を強調。

技選択フェーズ

「しんそく」「げきりん」「ほのおのパンチ」「りゅうのまい」から選択。 威力・命中・特殊効果の違いで戦略性が生まれます。

技説明画面

各技の説明テキストと配置により、見やすく情報を整理。 「戦闘へ戻る」でHP状態を保持して遷移可能な設計。

勝利画面

HPギリギリの状態で勝利した際に「ガブリアスに勝った！」と表示。 Messagebox演出で達成感を演出しています。

ゆびをふるモード

技が完全にランダムになるモード。発動技や効果、ダメージ量が毎ターン変わるエンタメ性の高い設計です。

バトルロジック

ターン制でプレイヤーと敵が交互に技を発動。技ごとに威力・命中・PPがあり、「りゅうのまい」で次の技のダメージが1.5倍になるなどの特殊効果も。 ゆびをふるモードでは約15種の技の中からランダムに発動され、戦略性よりも運と演出を重視した仕組みになっています。

制作情報

制作期間：1か月（2024年9月〜10月）
使用技術：Python / Tkinter / PIL / pygame / messagebox
担当範囲：全体設計・技ロジック・UI構成・BGM連携・画像演出
主な工夫：ターン制戦闘／技の個性付け／エンタメ性とUXの両立／著作権配慮（画像加工）
